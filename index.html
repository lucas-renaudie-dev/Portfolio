Game-Playing AI

An AI agent that plays the game of Reversi (Othello) using an Iterative-Deepening Alpha-Beta Pruning tree search, coded in Python. Features an evaluation function, based on a weighted sum of heuristic values. Implementation of a genetic algorithm to optimize the weights. Usage of search-space pruning and state memoization to avoid redundant computations. Placed 7th out of 150 in the tournament, with more than 99% win rate against real human players! See the report for more details on the agent.
